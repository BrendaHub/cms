## 欢迎使用 “师说CMS”

> 使用Java语言开发的CMS（使用Apache2协议，可免费用于商业用途）<br>
> Spring MVC为表现层，Spring为业务层，MyBatis为数据层。项目大量使用注解，代码注释清晰，文档齐全，是学习和二次开发的首选。<br>
